# 🐾 AdotaJá

Um sistema web desenvolvido para facilitar a **adoção responsável de animais**, conectando pessoas interessadas em adotar com tutores que desejam encontrar um novo lar para seus pets.

---

## 🌎 Sobre o projeto

O **AdotaJá** é um projeto acadêmico criado com o objetivo de ajudar a reduzir o abandono de animais na cidade de **Iguatu - CE**, promovendo a adoção consciente e responsável.

A plataforma permite:

- Cadastro de usuários
- Cadastro de animais para adoção
- Visualização de animais disponíveis
- Edição e gerenciamento dos próprios animais
- Contato direto com o tutor via WhatsApp

---

## 🚀 Funcionalidades

### 👤 Usuários
- Cadastro e login com autenticação
- Sessão persistente com Supabase

### 🐶 Animais
- Cadastro de animais com foto
- Upload de imagem
- Edição e exclusão
- Status:
  - Disponível
  - Em processo
  - Adotado

### 📱 Contato
- Botão de interesse
- Redirecionamento direto para WhatsApp com mensagem automática

### 🖼️ Interface
- Design moderno e responsivo
- Fallback com silhueta quando não há imagem
- Componentes reutilizáveis

---

## 🛠️ Tecnologias utilizadas

- ⚛️ React.js
- ⚡ Vite
- 🎨 CSS3
- 🧠 JavaScript (ES6+)
- 🗄️ Supabase (Auth + Database + Storage)
- 🎯 React Icons
